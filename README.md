#  Modern Portfolio Web Design

Welcome to my portfolio website repository! This project is designed with a modern, responsive, and dynamic architecture, combining the power of a component-based modern frontend with flexible backend integration.

---

## 🛠️ Tech Stack & Features

### **Frontend**
* **Next.js & React** – Core UI framework optimized for production and performance.
* **TypeScript** – Ensures robust, type-safe code to minimize runtime bugs.
* **Tailwind CSS** – Utility-first CSS framework for a clean, modern, and aesthetic design.
* **Shadcn UI / Components** – Accessible and customizable component library for a consistent layout.

### **Backend / Integration**
* **Python & Flask** – Utilized for managing dynamic data structures and template routing.

---

## 💻 Local Setup (How to Run on Your Machine)

Follow these steps to clone and run this project locally on your device:

### 1. Prerequisites
Make sure you have the following tools installed:
* [Node.js](https://nodejs.org/) (Latest LTS version highly recommended)
* [Python 3.x](https://www.python.org/)
* A package manager of your choice (`npm` or `pnpm`)

### 2. Clone the Repository
Open your terminal or command prompt, then run:
```bash
git clone [https://github.com/ArrasyEdlyn/Portfolio_Web_Design.git](https://github.com/ArrasyEdlyn/Portfolio_Web_Design.git)
cd Portfolio_Web_Design
```

###3. Frontend Setup (Node/Next.js)
Navigate to the project directory and install the required JavaScript/TypeScript dependencies:

#Using npm:
```bash
npm install react
```

Once the installation is complete, spin up frontend development server
```bash
npm run dev
```

###4. Backend SetUp(Python/Flask)
Open new terminal and install flask modul :
```bash
pip install flask
```

###Run Your Server
```bash
python app.py
```
Your Flask backend is now live and running at ```http://127.0.0.1:5000```.
